Projetos

- GG Music

Este projeto foi desenvolvido para a disciplina de Desenvolvimento de Sistemas Orientados a Objetos. O objetivo foi criar um sistema utilizando os princípios de orientação a objetos, incluindo herança, encapsulamento e polimorfismo. O projeto implementa uma rede social de músicas, onde os usuários podem compartilhar o que estão ouvindo com suas conexões, explorando recursos de interação e compartilhamento. Além disso, o código exemplifica o uso de padrões de projeto, persistência de dados e boas práticas de programação orientada a objetos. Todo o conteúdo está disponível no repositório.

- Site Jojos

Desenvolvido para formação como Desenvolvedor Front-end na Pixel, o projeto consiste em uma landing page mais blog de uma empresa de jogos retrô brasileira, Jojos. O site foi desenvolvido em ReactJS usando o framework Next.js, e posto em produção na Vercel. O blog, os jogos e as vagas deiponiveis foram 4integrados com o Sanity CMS.
